## 執筆ガイドライン

### 1. 文章量
**全体で2500-4000文字**
- セクション1（フック）: 短め
- セクション2（データ）: 長め
- セクション3（理由）: 中程度
- セクション4（CTA）: 短め

### 2. 文章スタイル

✅ **DO（推奨）**
- 友達に教えるような親しみやすさ
- 1文は50文字以内
- シンプルに、短く
- 数値は具体的に（「高い」→「+21%」）
- 問いかけを使う

❌ **DON'T（禁止）**
- 説教的な表現
- 同じ表現の繰り返し（2回まで）
- 専門用語の羅列
- 長い文章
- HTML/CSS/Markdown表の出力

---

### 3. 特殊マーカーの配置（重要）

以下の3つのマーカーを**正確な位置**に配置してください：

#### <CHART>
- **位置**: セクション2（2番目の##）の見出し直後
- **前後**: 必ず空行

**例**:
```
## セクション2の見出し

<CHART>

グラフをご覧ください...
```

#### <DATA_TABLE>
- **位置**: セクション2のグラフ説明の後
- **内容**: 最新年度のデータ（平均地価、価格帯、地点数、変動率、スコア）
- **前後**: 必ず空行

**例**:
```
...26年間の推移です。

<DATA_TABLE>

（出典：国土交通省 地価公示データ {latest_year}年）

注目すべきは...
```

#### <AFFILIATE>
- **位置**: セクション4の最後
- **前後**: 必ず空行

**例**:
```
...知ってみませんか？

<AFFILIATE>

まずは知ることから。
```

**禁止事項**:
- `![]()`形式の画像タグ
- `| 項目 | データ |`形式のMarkdown表
- HTML/CSSタグ
- マーカー以外の装飾

---

### 4. データの柔軟な解釈

データに応じて、最も説得力のあるストーリーを構築してください：

**価格帯の幅がある場合**:
「同じ町内でも○倍の差」→ 査定の必然性

**上昇トレンド**:
「市場の評価」「リーマン・コロナでも底堅い」

**減少トレンド**:
「だからこそ知るべき」「現状把握の重要性」

**横ばい**:
「安定した価値」「底堅さの証明」

---

### 5. 記事構成（3セクション）

#### セクション1: フック + 信頼性
1. 問題提起（読者の悩み）
2. 筆者の立場（独立系アナリスト）
3. データの信頼性（国土交通省、26年分）

#### セクション2: データ分析
1. <CHART>配置
2. グラフの説明（推移、主要イベント）
3. <DATA_TABLE>配置
4. 出典明記
5. データ解説（価格帯の幅、変動率）

#### セクション3: なぜ知るべきか
4つの理由を簡潔に：
1. 相続対策
2. 資産の見える化
3. 将来の選択肢
4. 金融機関との交渉

締めのメッセージ:
「今すぐ売る必要はない。でも知っておくべき」

#### セクション4: CTA
1. 問いかけ（価格帯の幅）
2. 行動への誘導
3. <AFFILIATE>配置
4. 最後の一押し

---

### 6. NG表現

❌ 「今売らないと損」「このままだと危険」（不安煽り）
❌ 「今が売り時」「高く売れるうちに」（売却推奨）
❌ 「容積率」「建蔽率」の詳細説明（専門用語）
❌ 「データを見ると」を3回以上（繰り返し）
