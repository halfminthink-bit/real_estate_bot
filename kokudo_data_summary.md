# 国土数値情報（地価公示）データ構造まとめ

## ディレクトリ: `data\raw\national\kokudo_suuchi`

### 概要
2000年から2025年までの26年分のデータが格納されています。
各年は `{YYYY}_13` フォルダに展開されています（13は東京都の都道府県コード）。

---

## 年別データ構造

### 【2000-2011年】古い形式
**フォルダ構造**: 直接配置（サブフォルダなし）

**ファイル形式**: Shapefile形式
- `KS-META-L01-{YY}_13-g.xml` - メタデータXML
- `L01-{YY}_13-g_LandPrice.dbf` - 属性データ
- `L01-{YY}_13-g_LandPrice.shp` - シェープファイル
- `L01-{YY}_13-g_LandPrice.shx` - インデックスファイル
- `L01-{YY}_13-g.xml` - データXML

**特徴**: 
- ファイル名に `-g` サフィックスと `_LandPrice` サフィックスが付く
- GeoJSON形式は含まれない

---

### 【2012-2014年】中間形式
**フォルダ構造**: 直接配置（サブフォルダなし）

**ファイル形式**: Shapefile形式
- `KS-META-L01-{YY}_13.xml` - メタデータXML
- `L01-{YY}_13.dbf` - 属性データ
- `L01-{YY}_13.shp` - シェープファイル
- `L01-{YY}_13.shx` - インデックスファイル
- `L01-{YY}_13.xml` - データXML

**特徴**:
- サフィックスが簡略化された
- GeoJSON形式は含まれない

---

### 【2015-2021年】GMLフォルダ形式
**フォルダ構造**: `{YYYY}_13/L01-{YY}_13_GML/` に配置

**ファイル形式**: Shapefile + GeoJSON
- `KS-META-L01-{YY}_13.xml` - メタデータXML
- `L01-{YY}_13.dbf` - 属性データ
- `L01-{YY}_13.prj` - 投影情報（2015年以降）
- `L01-{YY}_13.shp` - シェープファイル
- `L01-{YY}_13.shx` - インデックスファイル
- `L01-{YY}_13.xml` - データXML
- `L01-{YY}_13.geojson` - GeoJSON形式（2018年以降）

**特徴**:
- GMLフォルダ内に配置
- 2018年以降はGeoJSON形式も含まれる
- 2021年以降は `.gfs` ファイルも追加

---

### 【2022年】直接配置形式
**フォルダ構造**: 直接配置（GMLフォルダなし）

**ファイル形式**: Shapefile + GeoJSON
- `KS-META-L01-22_13.xml` - メタデータXML
- `L01-22_13.dbf` - 属性データ
- `L01-22_13.geojson` - GeoJSON形式
- `L01-22_13.gfs` - GFSファイル
- `L01-22_13.prj` - 投影情報
- `L01-22_13.shp` - シェープファイル
- `L01-22_13.shx` - インデックスファイル
- `L01-22_13.xml` - データXML

**特徴**:
- 唯一、GMLフォルダなしで直接配置されている年
- GeoJSONとGFSファイルを含む

---

### 【2023-2025年】最新形式
**フォルダ構造**: `{YYYY}_13/L01-{YY}_13_GML/` に配置

**ファイル形式**: Shapefile + GeoJSON
- `KS-META-L01-{YY}_13.xml` - メタデータXML
- `L01-{YY}_13.dbf` - 属性データ
- `L01-{YY}_13.geojson` - GeoJSON形式
- `L01-{YY}_13.gfs` - GFSファイル
- `L01-{YY}_13.prj` - 投影情報
- `L01-{YY}_13.shp` - シェープファイル
- `L01-{YY}_13.shx` - インデックスファイル
- `L01-{YY}_13.xml` - データXML

**特徴**:
- GMLフォルダ内に配置
- GeoJSONとGFSファイルを含む
- 最新の標準形式

---

## ファイル形式の変遷

| 期間 | フォルダ構造 | Shapefile | GeoJSON | GFS | PRJ |
|------|------------|-----------|---------|-----|-----|
| 2000-2011 | 直接配置 | ✓ | ✗ | ✗ | ✗ |
| 2012-2014 | 直接配置 | ✓ | ✗ | ✗ | ✗ |
| 2015-2017 | GMLフォルダ | ✓ | ✗ | ✗ | ✓ |
| 2018-2020 | GMLフォルダ | ✓ | ✓ | ✗ | ✓ |
| 2021 | GMLフォルダ | ✓ | ✓ | ✓ | ✓ |
| 2022 | 直接配置 | ✓ | ✓ | ✓ | ✓ |
| 2023-2025 | GMLフォルダ | ✓ | ✓ | ✓ | ✓ |

---

## ZIPファイル

各年のZIPファイルも `kokudo_suuchi` ディレクトリ直下に保存されています：
- `L01-00_13_GML.zip` ～ `L01-25_13_GML.zip`（全26ファイル）

---

## 注意事項

1. **フォルダ構造の違い**: 2015年以降と2022年でフォルダ構造が異なる
2. **ファイル形式の違い**: 2018年以降でGeoJSONが追加、2021年以降でGFSが追加
3. **ファイル名の違い**: 2000-2011年は `-g` と `_LandPrice` サフィックスが付く
4. **投影情報**: 2015年以降に `.prj` ファイルが追加
